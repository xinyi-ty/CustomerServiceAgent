企业客诉处理标准作业程序（SOP）

1. 客诉分级标准

根据客诉事件的业务影响和紧急程度，将客诉分为三个等级：

- L1（低紧急）：无安全风险，不影响产品基本使用，客户情绪平静。示例：缺失说明书、不会操作、咨询配件购买。响应时限：24小时内。责任处理人：一线客服。

- L2（中紧急）：影响部分功能，有潜在风险但尚未发生，客户情绪不满。示例：核心部件异响、批次性缺陷、软件频繁卡顿。响应时限：4小时内。责任处理人：部门经理。

- L3（高紧急）：涉及人身或财产安全，或可能引发群体性投诉，客户情绪激烈。示例：设备冒烟起火、漏电、严重伤人风险、大批量同问题投诉。响应时限：30分钟内。责任处理人：总经理或技术总监。

2. 硬件类故障处置细则

2.1 设备冒烟、起火、烧焦味（L3）
判定关键词：冒烟、起火、烧焦、火花、刺鼻味、爆炸、烟。
自动回复话术：
【紧急安全警告】请立即切断设备电源，并远离设备至少2米。如果已经起火，请使用干粉灭火器，并拨打119。系统已自动通知我司技术总监，他将在15分钟内电话联系您提供进一步指引。请勿尝试自行拆解或移动设备。
处理动作：生成高紧急工单，推送至总经理看板，标记为“安全事件”，24小时内出具事故分析报告。

2.2 漏电、触电（L3）
判定关键词：漏电、触电、麻麻的、电击、shock。
自动回复话术：
【紧急】请立即停止使用设备并拔掉电源插头。不要接触金属部分。我们的安全工程师会立刻联系您。如果已受伤请及时就医。
处理动作：同2.1，且需登记人员受伤情况。

2.3 核心部件损坏（L2）
判定关键词：主轴断裂、电机不转、控制板无响应、齿轮打滑、异响严重、无法启动、马达坏。
自动回复话术：
非常抱歉给您带来不便。您反馈的设备核心部件问题已引起我们重视。我们将安排资深工程师与您预约远程诊断，若确认硬件故障，我们会优先寄送替换部件并提供技术指导更换。请保持电话畅通。
处理动作：工单派发至部门经理，经理需2小时内联系客户，安排上门或返厂维修。

2.4 外壳破损、外观划痕（L2）
判定关键词：外壳裂了、屏幕破碎、划痕、缺角、变形、破裂。
自动回复话术：
对于外观损伤我们深表歉意。请您上传多角度清晰照片，我们的质量部门会分析原因。若确属出厂问题，我们将免费为您更换外壳并补偿优惠券。换货流程约3-5个工作日。
处理动作：部门经理判定是否换新，并协调物流。

2.5 配件缺失、损坏（L1）
判定关键词：少螺丝、缺少数据线、没有说明书、包装内缺件、缺配件。
自动回复话术：
您好，我们很重视您提到的配件缺失问题。请访问以下链接填写补发申请：https://example.com/replacement 。通常3个工作日内免费寄出。您也可以联系在线客服提供地址，我们会优先处理。
处理动作：一线客服直接操作补发，无需升级。

3. 软件/固件类问题

3.1 频繁死机、卡顿（L2）
判定关键词：死机、卡死、无响应、系统崩溃、蓝屏、卡顿。
自动回复话术：
系统卡顿可能是软件冲突或固件bug导致。请您尝试升级至最新固件版本（下载链接：https://example.com/firmware），并重启设备。若问题依旧，请提供设备日志（导出教程见附件），我们会安排软件工程师分析。
处理动作：若超过5人反馈相同问题，自动升级为L2并通知产品经理。

3.2 无法连接网络/蓝牙（L2）
判定关键词：连不上WiFi、蓝牙搜不到、配对失败、连接超时。
自动回复话术：
请检查您的路由器是否开启2.4GHz频段（本设备不支持5GHz），并尝试断电重启设备。如果仍无法解决，请告知我们您的路由器型号，我们将提供兼容列表。
处理动作：一线客服尝试基础排障，若失败则转技术部门。

3.3 操作疑问（L1）
判定关键词：怎么使用？哪里下载APP？如何注册？教程、说明书、不会用。
自动回复话术：
您可访问说明书电子版：https://example.com/manual 或观看教学视频：https://example.com/video。也可在聊天框输入“新手引导”获取步骤。
处理动作：无需人工介入。

4. 物流/售后服务类

4.1 运输造成损坏（L2）
判定关键词：快递破损、外包装压坏、货物变形、运输损坏、物流弄坏。
自动回复话术：
很遗憾听到您的商品在运输中受损。请您保留原包装并拍摄破损照片（包括外箱六面），我们会联系物流公司理赔。同时我们会为您安排换货，新货将优先发出。
处理动作：部门经理审批换货，同步发起物流投诉。

4.2 保修期内故障（L2）
判定关键词：保修期内、刚买不久、购买日期、SN码校验在保、在保修期。
自动回复话术：
经系统查询，您的设备尚在保修期内。您可以选择：① 到附近授权维修点免费维修；② 寄回工厂检测（运费我方承担）。请提供您的地址和方便时间，我们会安排快递上门取件。
处理动作：系统自动校验SN码后，若在保则推送维修方案。

4.3 过保维修咨询（L1）
判定关键词：过保、保修期过了、维修费用、过保修期。
自动回复话术：
您的设备已超出保修期。我们提供付费维修服务，维修报价单链接已发送至您注册邮箱。您也可联系客服获取当地维修点信息。感谢理解！
处理动作：一线客服发送标准化报价单。

5. 群体性/舆情风险事件

5.1 同批次多起相同故障（L3）
判定关键词：多个用户反馈、批次号相同、批量问题、集体投诉。
自动回复话术：
我们已经关注到您反馈的问题，并已成立专项小组进行调查。请您放心，我们会尽快公布解决方案。在此期间，您可申请备用设备（若适用）。
处理动作：自动推送至总经理看板，启动公关预案，停止该批次发货。

5.2 媒体/社交平台曝光（L3）
判定关键词：微博曝光、抖音投诉、央视报道、媒体、热搜、大V。
自动回复话术：
我们已注意到您的公开反馈，并高度重视。我们的公关经理将在1小时内与您联系。为加快处理，请您私信提供订单号和联系方式。
处理动作：最高优先级，由总经理直接对接。

6. 常见问题快速处理表

| 客户表述示例 | 紧急度 | 处理建议 |
|------------|--------|----------|
| “机器冒烟了” | L3 | 立即断电，通知技术总监，出具安全报告 |
| “少了一个螺丝” | L1 | 提供补发链接，登记地址 |
| “电机异响，声音很大” | L2 | 安排工程师远程诊断，准备替换部件 |
| “外壳摔裂了” | L2 | 要求提供照片，安排换壳或换新 |
| “连不上WiFi” | L2 | 指导2.4G切换，重启路由器 |
| “怎么开机？” | L1 | 发送电子说明书链接 |
| “快递箱子被压扁了” | L2 | 登记物流理赔，安排换货 |
| “刚买三天就坏了，在保修期吗？” | L2 | 校验SN码，推送免费维修方案 |
| “过了保修期修要多少钱？” | L1 | 发送报价单链接 |
| “我要去微博曝光你们” | L3 | 公关经理1小时内联系 |
| “我买的这一批次好多人都坏了” | L3 | 成立专项小组，停止发货 |

备注：所有客诉处理完毕后，需在系统中关闭工单，并记录解决方案。对于L3级别事件，须在24小时内完成复盘报告并归档。
