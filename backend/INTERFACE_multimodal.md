# Multimodal Interface Specification (OCR + Warranty)

Author: multimodal
Date: 2026-05-12

---

## 1. Module Responsibilities

### OCR (ocr_utils.py)
- Extract text from image/video
- Output: raw string only
- Does NOT perform structured extraction

### Warranty (warranty.py)
- Determine warranty status from SN code
- Independent logic (not from LLM)

---

## 2. Function Interfaces

### OCR

extract_text_from_image(image_bytes: bytes) -> str

- Input: image bytes
- Output: string (OCR text)
- If no text: return ""

---

extract_text_from_video(video_bytes: bytes) -> str

- Input: video bytes
- Output: OCR text from first frame
- If failed: return ""

---

### Warranty

check_warranty(sn_code: str) -> str

Return values:
- "In_Warranty"
- "Out_of_Warranty"
- "Unknown"

---

## 3. Data Flow

User Input (text + image/video)
        ↓
OCR (text)
        ↓
LLM (structured extraction)
        ↓
extracted_data.sn_code
        ↓
Warranty check
        ↓
Main assembles final ticket

---

## 4. LLM Output Format (Required)

LLM must return:

{
  "issue_category": "string",
  "business_impact": "Safety_Hazard | Production_Stop | Normal",
  "urgency_level": "Low | Medium | High",
  "routing": "frontline_worker | manager_dashboard | general_manager_dashboard",
  "reply": "string",
  "extracted_data": {
    "order_id": "",
    "model_number": "",
    "batch_code": "",
    "sn_code": ""
  }
}

---

## 5. Important Constraints

- SN field MUST be named: sn_code
- Do NOT use: sn / snCode / serial

- OCR returns ONLY string
- No dict / list output

- Warranty is NOT generated by LLM

- If sn_code is empty → return "Unknown"

---

## 6. Final Ticket Assembly (Handled in main.py)

{
  "ticket_id": "...",
  "extracted_data": {...},
  "agent_business_assessment": {
    "issue_category": "...",
    "business_impact": "...",
    "urgency_level": "...",
    "warranty_status": "..."
  },
  "routing_decision": "...",
  "auto_reply_sent": "..."
}

Note:
- warranty_status comes from warranty.py
- NOT from LLM

---

## 7. Current Implementation

- OCR: PaddleOCR
- Video: first frame extraction
- Warranty rule:
  - Prefix 24 / 25 → In_Warranty
  - Others → Out_of_Warranty
